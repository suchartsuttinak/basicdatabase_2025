คำสั่ง
ช่วยปรับหน้านี้ให้มีมาตรฐานเดียวกับหน้าที่คุณเคยทำให้ผมก่อนหน้า
โดยยึดหลักดังนี้

1) ใช้โครงสร้าง UI แบบ modern / clean / international standard
2) ต้อง responsive ทุกขนาดหน้าจอ
3) มือถือและจอเล็กต้องใช้งานได้จริง และต้องมี scroll ใน modal/body ถ้าจำเป็น
4) ป้องกันปัญหาเดิมที่เคยเจอ:
   - modal จอเล็กแล้วไม่มี scroll
   - ฟอร์มชิดซ้ายหรือผิดสัดส่วน
   - ปุ่มไม่เรียงในแถวเดียว
   - ปุ่มซ้อนกันหรือขนาดไม่เท่ากัน
   - radio / checkbox card ตัดบรรทัดไม่สวย
   - table ล้นจอโดยไม่มี scroll
   - create และ edit หน้าตาไม่เหมือนกัน
   - CSS ชนกัน หรือ selector ทับกันมั่ว
5) ให้ใช้โครงสร้าง create/edit มาตรฐานเดียวกัน
6) ถ้ามี modal:
   - modal-content ต้องจัด layout ให้ถูก
   - form ภายใน modal ต้องเป็น flex column
   - modal-body ต้อง scroll ได้บนจอเล็ก
   - footer ต้องไม่ลอยทับเนื้อหา
7) ถ้ามี action buttons:
   - desktop ให้อยู่แถวเดียวกัน
   - mobile ถ้าพื้นที่ไม่พอให้ scroll แนวนอนหรือเรียงลงอย่างสวยงามตามความเหมาะสม
8) ถ้ามี radio-card หรือ option-card:
   - ความกว้างสมดุล
   - ความสูงใกล้เคียงกัน
   - ข้อความไม่แตกผิดรูป
9) ขอให้คุณ:
   - วิเคราะห์ปัญหาก่อน
   - ปรับ HTML + CSS + JS ที่จำเป็นให้ครบ
   - ส่งโค้ดที่พร้อมใช้งานได้ทันที
   - อย่าทำให้กระทบหน้าอื่น
   - ถ้าต้อง override CSS ให้ทำอย่างปลอดภัย
10) ช่วยจัดโครงสร้าง class ให้ดูแลง่าย และลดปัญหาซ้ำในอนาคต

นี่คือโค้ดของหน้า:
[วาง Blade / HTML / CSS / JS ที่เกี่ยวข้อง]

คำสั่งแบบสั้น

ถ้าคุณอยากพิมพ์สั้น ๆ ใช้อันนี้ได้

ช่วยปรับหน้านี้ให้สวยงามแบบเดียวกับหน้ามาตรฐานที่คุณเคยทำให้ผม
เน้น responsive, modal scroll บนจอเล็ก, ปุ่มไม่ซ้อน, create/edit ใช้มาตรฐานเดียวกัน, ไม่ให้ CSS ชนกัน, และส่งโค้ดพร้อมใช้งานทั้งหมด

คำสั่งแบบละเอียดที่สุด

อันนี้เหมาะเวลาคุณส่งงานหลายหน้า
ช่วยรีแฟกเตอร์หน้านี้ทั้งระบบให้เป็นมาตรฐานเดียวกับหน้าที่คุณเคยออกแบบให้ผมก่อนหน้า
ต้องตรวจทั้ง Blade, CSS, JS และ responsive behavior

สิ่งที่ต้องตรวจและแก้:
- layout desktop / tablet / mobile
- modal structure
- modal scroll บนจอเล็ก
- form spacing
- button alignment
- table responsiveness
- radio / checkbox card layout
- create/edit consistency
- selector conflict
- overflow / wrapping / width issue
- footer ทับ content
- class naming ให้เป็นระบบ

ขอผลลัพธ์เป็น:
1. วิเคราะห์ปัญหาสั้น ๆ
2. โค้ดที่แก้แล้วทั้งหมด
3. ใช้งานได้ทันที
4. ไม่กระทบหน้าอื่น

เวอร์ชันโปร (ใช้ตอนงานใหญ่ / หลายปัญหา)

อันนี้คือ “สูตรเต็ม” ที่จะได้งานเนียนที่สุด

ช่วยรีแฟกเตอร์หน้านี้ให้เป็นมาตรฐานเดียวกับหน้าที่คุณเคยออกแบบให้ผมก่อนหน้า

Requirement:
- UI ต้อง modern / clean / international standard
- responsive ทุกหน้าจอ (desktop / tablet / mobile)
- mobile ต้องใช้งานได้จริง

ตรวจและแก้:
- modal structure + scroll บนจอเล็ก
- form layout และ spacing
- button alignment (ไม่ซ้อน / ขนาดเท่ากัน)
- table responsive + scroll
- radio / checkbox card layout
- create/edit consistency

- overflow / width / flex issue

ข้อห้าม:
- ห้ามทำให้หน้าอื่นพัง
- ห้ามแก้แบบ patch มั่ว ต้องเป็นโครงที่ถูกต้อง

ขอผลลัพธ์:
1. วิเคราะห์สั้น ๆ
2. โค้ดที่แก้แล้วทั้งหมด (HTML + CSS + JS)
3. ใช้งานได้ทันที
